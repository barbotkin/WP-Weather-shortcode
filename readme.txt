== Description ==

The plugin displays the weather shortcode. Shortcode you can use anywhere in the site: pages, records, widgets, in function.php or in the other file.

== Installation ==


1. Upload 'Weather Shortcode' to the '/wp-content/plugins/'' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings -> Weather shortcode.
4. Use the shortcode :)

== Frequently Asked Questions ==

= How to use shortcode? =

Shortcode: [ms-weather]. Use php: '<?php echo do_shortcode("[ms-weather]""); ?>'

= How to change the city in the weather? =

Go to settings -> Weather shortcode.
